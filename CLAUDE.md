# ali-chat

Next.js app (React 19, TypeScript, Tailwind 4). Agent skills from [mattpocock/skills](https://github.com/mattpocock/skills) live under `.agents/skills/`.

## Agent skills

### Issue tracker

Issues and PRDs live as markdown under `.scratch/` (no Git remote on this workspace yet). See `docs/agents/issue-tracker.md`.

### Triage labels

Default canonical role strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` and `docs/adr/` at repo root when present. See `docs/agents/domain.md`.
